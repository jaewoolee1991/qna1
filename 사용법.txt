====================================
📷 현장 사진 업로드 앱 - 간단 사용법
====================================

🎯 시작하기
-----------
1. upload.html 파일을 모바일 브라우저에서 열기
2. 작업 정보 입력
3. 사진 4장 선택
4. 업로드 버튼 클릭
5. gallery.html에서 확인

📱 필수 단계
-----------
1단계: 작업 정보 입력
  ✓ 작업자 이름 (필수)
  ✓ 프로젝트명 (필수)
  - 작업 메모 (선택)
  - 위치 정보 (자동 감지됨)

2단계: 사진 선택 (4장)
  ✓ "📷 사진 촬영/선택" 버튼 클릭
  ✓ 카메라로 촬영 또는 갤러리에서 선택
  ✓ 미리보기에서 확인
  ✓ 불필요한 사진은 ❌ 버튼으로 삭제

3단계: 업로드
  ✓ 4장이 모두 선택되면 버튼 활성화
  ✓ "⬆️ 사진 업로드하기" 클릭
  ✓ 진행률 확인
  ✓ 완료 후 자동으로 갤러리로 이동

🖼️ 갤러리 사용법
--------------
- 프로젝트, 작업자, 날짜로 필터링
- 카드 클릭하여 상세보기
- 사진 클릭하여 확대 보기
- 삭제 버튼으로 업로드 삭제

📍 위치 선택 (선택사항)
--------------------
- 자동으로 현재 위치가 감지됩니다
- "📍 위치 선택" 버튼으로 수동 선택 가능
- 지도에서 클릭하여 위치 선택
- 주소가 자동으로 표시됩니다

⚠️ 주의사항
----------
✓ 정확히 4장의 사진만 업로드 가능
✓ 파일 크기: 각 10MB 이하
✓ 이미지 파일만 업로드 가능
✓ 작업자 이름과 프로젝트명은 필수
✓ 인터넷 연결 필요

🔧 Firebase 설정 필요
-------------------
Firebase Console에서:
1. Firestore Database 활성화
2. Firebase Storage 활성화
3. 보안 규칙 설정 (README 참조)

📱 권장 브라우저
--------------
✓ Chrome (Android/iOS)
✓ Safari (iOS)
✓ Samsung Internet
✓ Edge (Mobile)

🆘 문제 해결
-----------
Q: 업로드가 안 돼요
A: Firebase 설정과 인터넷 연결 확인

Q: 위치가 안 잡혀요
A: 브라우저 위치 권한 확인, 수동 선택 사용

Q: 카메라가 실행 안 돼요
A: 브라우저 카메라 권한 확인, HTTPS 연결 확인

Q: 갤러리가 안 보여요
A: Firestore 규칙 확인, 페이지 새로고침
